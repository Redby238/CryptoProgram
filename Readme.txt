So in this program was implemented functionality like:
Multi-page application with navigation,
The main page displays the top 10 currencies that were returned by API,
Page with the ability to view detailed information about the currency: price, volume, in which markets it can be purchased and at what price,
Possibility of searching for currency by name or code. Also if you want to reset the page you delete the word in the search field and press one more time to reset it
Page in which you can convert one currency to another.
From technologies was used
WPF
C#
Newtonsoft.Json
HttpClient
